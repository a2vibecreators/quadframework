## Minimal Setup - Gemini API Key Only

**Total Setup Time: 3 minutes 30 seconds**

### Step 1: Get Gemini API Key (2 minutes)
1. Go to: https://ai.google.dev/gemini-api
2. Click "Get API Key"
3. Sign in with Google account
4. Click "Create API Key"
5. Copy key (starts with `AIza...`)

### Step 2: Add to QUAD Platform (30 seconds)
1. Go to: https://quadframe.work/settings
2. Paste Gemini API key in "AI Provider" field
3. Click "Save"
4. Done!

### Step 3: Build First Website (1 minute)
1. Go to: https://quadframe.work/generate
2. Type: "Personal website with home page and contact form"
3. Click "Generate & Deploy"
4. Wait 35 seconds
5. Website LIVE at: yourname.quadframe.work

**NO credit card, NO hosting setup, NO domain config needed!**
