# QUAD Projects

Demo projects built using QUAD methodology.

## Structure

Each project contains:
```
project-name/
├── .quad/
│   └── project.json      # Project config (from quad-init)
└── project-setup.xlsx    # Source Excel (editable)
```

Other files are GENERATED by QUAD commands:
- `quad-plan` → `.quad/features/`
- `quad-write` → `src/`
- `quad-doc` → `docs/`
